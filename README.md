## Installation
### PNPM
> On Linux or macOS:

`curl -fsSL https://get.pnpm.io/install.sh | sh -`

> On Windows (PowerShell):

`iwr https://get.pnpm.io/install.ps1 -useb | iex`

### Dependencies
`pnpm install`

## Development
Run following scripts
```
pnpm run decorators

pnpm run metadata
```
